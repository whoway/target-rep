        # 2.LaTex排版

        • 有哪些好的 LaTeX 编辑器？ (opens new window)

        • LyX (opens new window)，所见即所得，有点类似Word，但是可以编辑LaTeX源码。可以用来先搭建个大概，比较快。

        • Visual Studio Code (opens new window)，微软出的免费文本编辑器，和 Atom、Sublime Text 操作类似，文本编辑、日常开发等都能干。配合扩展（LaTeX-Workshop (opens new window)），可编译、可反定向、有代码提示。

        • 只推荐 Overleaf (opens new window)（在线）和 BaKoMa Tex (opens new window)（本地）。

        • 新手只推荐 Overleaf。下附实际使用截图。『我用的谷歌账号』

          • Overleaf 的帮助文档是如此的齐全，以至于在站内搜索就能给出几乎所有答案。快开写吧，现在马上！
          • 用线上的方式的好处是，不需要搭建环境！避免搭建环境的各种坑

        除了本科毕业论文用的Word，之后就一直用LaTeX了。起初的原因是硕士时期的教授强烈推荐（要求）的，当时看了一眼，觉得我去好复杂呀，就心生抗拒，然后教授就威胁我说“在顶级的国际会议和国际期刊用word投论文是不会有人看的，直接拒掉的会！”，当时一听吓尿了（哎，too young too naive）,不过LaTeX着实适合写论文做学术，因为：

        • 1， 其实写一篇论文的量并不大，而且周期不短，但是要修修改改反反复复N多次，所以，LaTeX比word在写的时候多出来的复杂度完全不是问题，但是，==每次对某一个地方修改了一下，word的格式就全乱了，就得再调一次格式带来的不便性是绝对要人命的==！！！

        • 2， 而且，写的时候，Latex下想怎么写就怎么写，想为了看着清晰明了一目了然就每段空个几行，但是在生成页面是完全看不出来的，所以，写起来也是很洒脱的很受用的

          1. 不得不说，LaTeX写出来确实漂亮

        所以，LaTeX就是好用舒心不折腾的宝钗，word就有些像黛玉了，得小心供着用！！！

        • 答主下的什么软件

        • WinEdt + Miktex, 软件很多，随意选择便可。

        # 3.经常用 LaTeX 的=人？

        1。数学物理类的研究人员，排版数学什么的必须要这个 2。某些杂志的排版员，如《物理学报》，记件算工资 3。对word有排版恐惧症的人，如版本问题等 4。某些排版小公司，例如北京的一些，排国际论文捞钱的 5。对Knuth有滔滔崇敬之情的，拿这个当爱好 6。纯属折腾的

        • 我在英国读经济学硕士，所有老师上课用的notes和ppt（beamer）都是用LaTeX写的，他们自己写paper写cv也是用LaTeX，平常看journal或者一些学者的working paper也都是LaTeX排的版，所以我觉得在西方学术界应该是比较普及的吧。

        • 硕士期间写了3篇coursework，用的word，期间各种糟心，改样式，公式编号简直恶心人，插图插表也永远达不到理想效果，一改就乱，我不动敌不动，我一动敌乱动。因为打算申博，硕士尾声学了LaTeX，写了一篇review，现在在写毕业论文，唯一的感想就是太爽了，有一种coding的快感，为什么没有早点用上，那排版简直太美，太省心了，让你只专注在内容上。

        • 下面分享一些我入门的小经验，我大约花了一天时间去学。我觉得要精通LaTeX是很难的，但是写写论文简直太快上手了，边用边学才是王道，我至今没遇到20分钟解决不了的问题。

          Overleaf：我的主力编辑器，之前是用texmaker，其他编辑器我接触不多，但接触了几个都比较反人类。这个的优点是什么都不用安装，方便跨平台（甚至ipad，对你没看错，ipad瞬间变生产力）或者多人工作，自动同步，不用另外搞网盘u盘，有很多templet，代码自动补全；缺点是没网就歇菜了。

        做表：随便google一搜latex table就可以可视化做好再复制代码进去。

        latex help：ios的app，可能安卓也有类似的，方便查各种代码，必备。

        Bibtex配合google scholar：业界良心啊，做bibliography简直不要太方便，要用的文章全部加到library之后一键导出bibtex，还可以分标签管理，学会的时候我哭了。

        教程：我看的是youtube的Michelle Krummel的，一上午就能看完，讲得很清楚。其实教程不用全部看完的，挑自己需要的学会了就可以了，比如光margin她就教了一堆方法，我学了一个geometry的就跳过了。
