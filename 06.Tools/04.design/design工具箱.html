          # 2.剪映

          • 剪辑最常用6个步骤
          • 粗剪
            • 导入素材，内置了一个很强大的素材库。
            • 可以精确到帧剪辑
            • 加转场，剪映提供了很多
              • 比如，推进，放大缩小之类的转场。
          • 加包装
            • 就是给画面增添一些元素
            • 好看的文字，加一下片头，人名头啊
            • 「有很多专业做剪辑的人就是这些样子」
          • 调色
            • 滤镜
          • 处理一下音频
          • 加字幕
          • 导出

          # chrome截长图

          • 1.按F12打开开发者工具
          • 2.按CTRL+SHIFT+p打开命令行
          • 3.输入full，点击Capture full size screenshot命令，会自动截截取当前页面的所有内容，生成图片，并且下载
