      # 源码阅读工具

      • https://elixir.bootlin.com/

      ​ 是一个在线的 Linux 内核代码浏览器，它提供了方便的在线查看 Linux 内核源代码的功能，可以查看各个版本的 Linux 内核代码、源码注释以及相关的文档。这个网站是由一个名为 Bootlin 的咨询公司创建和维护的，旨在为 Linux 内核开发人员提供方便的代码查看和参考工具。它是免费的，并且是开源的，其代码托管在 GitHub 上
