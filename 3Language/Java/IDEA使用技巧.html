        # 1.基本注意事项

        • 类名要和文件名一致

        • 文件里面只能由一个public类

        # 2.java有没有类似C++的万能头文件？

        • 没有万能
        • 在IDEA中，你输入相关类会后可以按Alt+enter导入
        • java一般不需要关注导入
        • 可以自动导入

        # 3.快捷键

        # main函数

        • 直接打main+tab
        • 或者psvm+tab

        # 输出

        • sout+tab
        psvm main 函数

fori for循环

sout 输出

shift+f6 重命名

ctrl+J 显示所有快捷键模板

Alt+Insert 生成构造函数、toString、getter/setter、重写父类方法等
Ctrl+Y 删除行
Ctrl+D 复制行
Ctrl+</> 折叠代码
Alt+Insert 可以新建类、方法等任何东西
Ctrl+Alt+O 格式化import列表
Ctrl+Alt+L 格式化代码
Ctrl+Alt+T 创建单元测试用例
Alt+Shift+F10 运行程序
Shift+F9 启动调试
Ctrl+F2 停止

Crtl+N：定位到Java类
Crtl+Shift+N：定位到文件
Crtl+Alt+Shift+N：定位到标识，这里标识主要是指Java类中的属性和函数名

F9 resume programe 恢复程序
Alt+F10 show execution point 显示执行断点
F8 Step Over 相当于eclipse的f6 跳到下一步
F7 Step Into 相当于eclipse的f5就是 进入到代码
Alt+shift+F7 Force Step Into 这个是强制进入代码
Shift+F8 Step Out 相当于eclipse的f8跳到下一个断点，也相当于eclipse的f7跳出函数
Atl+F9 Run To Cursor 运行到光标处
ctrl+shift+F9 debug运行java类
ctrl+shift+F10 正常运行java类
alt+F8 debug时选中查看值
