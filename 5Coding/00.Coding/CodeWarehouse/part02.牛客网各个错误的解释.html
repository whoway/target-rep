          # 1.各大OJ常见状态

          状态 常见情况
          CE（Compile Error）编译错误 1、比如C++11用的C++98编译的 2、C++用的python编译的
          RE（Runtime Error）运行时错误
          PE（Presentation Error）格式错误 多输出空格或换行
          TLE（Time Limit Exceeded）运行超时 1、某种数据导致死循环 2、算法复杂度高了
          MLE（Memory Limit Exceeded）内存超限 用了太多空间，如，数组开太大了
          OLE（Output Limit Exceeded）输出超限
          WA（Wrong Answer）
          AC（accept）

          # 2.RE（Runtime Error）运行时错误

          常见

          • 1、段错误（直接原因：非法访问内存，比如，数组越界、指针乱指）
          int a[5];
cout<<a[5]<<endl;//典型的数字越界
          1
          2

          • 2、浮点错误，您的程序运行时发生浮点错误，比如遇到了除以 0 的情况，比如，除数为0，模数为0 case通过率为0.00%

          • 3、递归爆栈（一般由于递归时层数过深导致）

          # 3.OLE输出超限

          程序输出了过量的内容（一般是指数量非常多）

          • 1、输出大量调试信息
          • 2、特殊数据导致的死循环输出？（此处）
