        # 论文写作

        # 1.文章具体写法

        • 综述要归类
          • 综述的，每个老师口味不一样。
        • 标题
          • 标题不要太大，是单面
          • 相关内容
          • 使用的方法
        • 结果分析，不能写，请见表格。必须你自己写出来！别人不会去其中
        • 如果你的全文只出现某个自创的缩写3-5次，就不要自创。除非是其他地方的缩写。
        • 公式是融入到句子中的，前面有冒号之类的，或者后面有,或者.啥的。

        # 2.期刊的要求

        • 如果目标期刊要求是：7页，那必须7页
        • 目标期刊是什么格式就用什么格式，比如双面还
        • 超过7页，直接不会被审稿的
        • 一般标题越短，引用越多，但是也可以写很清楚的长的

        # 3.英语单词

        一般不会写：drawback

        • TeX的排版

        • and so on，平常好的文章没见到。

        • include/such as，差不多就是不全部枚举，不需要加等等

        • 有的老师可能不会用PDF去改，只会有Word去改。

        • for和of在标题中不用大写！

        • 参考很容易出错，并不是用来Endnote就没有什么问题，要检查。

          • 如果文章是“预印版”就很麻烦，要检查（不要是网站下载的那种），一定要找一下那些最新的！
          • PDF可以用grammar进行，单数复数检查啥的（这样老师就会给你少
          • 尽量别用生僻的单词，当然可以换单词。
