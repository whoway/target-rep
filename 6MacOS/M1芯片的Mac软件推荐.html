# 5.1.开发者

  • item无疑是最好用的终端
    • 搭配上，on my zsh
    • 这个可以里面使用profile等方式，连接公司堡垒机，非常好用【比如百度】
  • mac中，大家一定要记得安装一个软件，叫homebrew
    • 轻松管理和安装
  • dash
    • 是快捷的文档查询软件
    • 很方便
  • 虚拟机
    • 鱼皮安装的，Parallels Desktop （但是要钱）
      • 他用过Mac上最好的虚拟机了，刚装虚拟机的时候，他就会自动给你安装Windows11
    • Virtual Box
      • 关键免费，但是没上面的方便
  • PicGo
    • 上传图片
    • utools
  • EasyConnect，深圳鹏城
  • clash，翻墙
  • mac上用Automator编写自动脚本，类似按键精灵？
    • https://blog.csdn.net/destiny_ac/article/details/43965909#
  • Filezilla
