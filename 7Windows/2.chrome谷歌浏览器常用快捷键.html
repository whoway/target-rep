              # 背景知识-什么是浏览器的窗口和标签页[标签]？

              • 窗口：就是每个网址或链接都打开一个新的浏览器界面。

              • 选项卡：又叫标签，是在同一个浏览器界面同时打开多个页面，用标签的形式显示，点击不同标签就可以切换到不同页面。

              # 1.个人常用快捷键

              快捷键 作用
              ctrl+w Withdraw，移开，退出 关闭当前的标签页【最常用】
              Alt+F4 关闭当前chrome浏览器的窗口
              Ctrl+J 查看“下载”页
              Ctrl+D 将当前网页 加入书签？【但是我还没找到好方法如何取消收藏？】
              Ctrl+F Find，查找 在当前这个网页中查找你的文字【类似word的方式】
              Ctrl+T Table，标签 新建标签页【可能T代表的tab，标签页】
              Ctrl+L或Alt+D Link，链接 选中浏览器的【网址区域的内容】，你就能直接在那输入你要搜索的东西了
              Ctrl+H History，历史 查看“历史记录”页
              Ctrl+P Print/er，打印/打印机 打印当前页【比如：你可以使用这个启动打印这个网页为pdf】
              Ctrl+S Save，保存 保存当前页【一般是搞前端的同学，想要保存这个网页的html啥的】
              F5 重新加载当前页
              Ctrl+F5 【去除静态缓存】重新加载当前页，重新加载当前页，但是忽略缓存内容（『自从使用了静态博客，对此理解更深了，这是缓存刷新』）
              ESC 取消加载当前页【也就是不让这个网页转圈圈了】】

              # 2.次常用

              • Ctrl+Shift+N 新建是无痕浏览模式的窗口（可以用于减少电脑缓存）

              • Ctrl+O 是用该浏览器打开电脑上的东西

              • 按住Alt+鼠标左键点击链接： 下载链接

              • Alt+D或者Ctrl+L 选中网址区域中的内容（效果和F6差不多一样——注意，F6和TAB差不多会慢慢走动，所以准确来说Alt+D才是最准确的），还可以配合一个：接着键入网址，然后按 Alt+Enter 键在新标签页中打开网址

                • 建议使用『alt+D』，而不是F6这样的
                • 原因：比如在这个电脑上改名的快捷键是F2，但是别的笔记本上可能就是『Fn+F2』，不是统一的，但是上面的不一样

              • Alt +RIGHT ARROW（其实就是→）：显示前一页（前进键）

              • Alt +LEFT ARROW（其实就是←）：显示后一页（后退键）

              • Ctrl+shit+DEL 调出界面—删除浏览记录

              • Ctrl+shift+T 重新打开上次关闭的标签页，特别值得一谈的是Google可以记住最近关闭的10个标签页

              • Ctrl+N 新建窗口（和Ctrl+T有些不同）窗口似乎是一个新的进程，而标签页是一个进程中的其他的？

              • 按住Ctrl点击链接 从后台在新标签页中打开链接，但你仍然停留在（当前）标签页中

              • 按住Ctrl+shift点击链接 在新标签页中打开链接，同时切换到（新打开）的标签页

              # 3.互联网web前端和后端常用快捷键

              • Shift+ESC 查看浏览器的“任务管理器”——类似于电脑的“任务管理器”的东西
                • 一般是互联网的web前端和后端才使用
              • Ctrl+U 查看网页源代码（前端代码用）

              # 4.其他快捷键

              • 一系列快捷键：Ctrl+1到8可以直接切换到指定位置编号的标签页，你按下的数字就代表那个相应的标签位置。
                • Ctrl+9切换到最后的标签页面
                • CTRL+TAB：在页面上的各框架中切换（加shift反向）
              • 地址栏中，键入www.和.com之间的部分，然后按Ctrl+enter会自动，为你在地址栏中的内容添加www.和.com
              • 3）地址栏中 键入网址，然后按Alt+enter 在（新标签页）中打开该网站
              • F3 查找与你“在网页上查找”输入框总匹配的下一项
                shift+F3查找与你“在网页上查找”输入框总匹配的上一项
