# 💎Ruby

​ ruby，n. 红宝石。1

​ ruby是一种面向对象的动态脚本语言

​ Rails是使用纯ruby编写的框架(framework)。它对web开发提供了强有力的支持，如支持数据映射、MVC模式、Web Services、安全等。而且这些功能操作起来要比同类的产品容易的多，如MVC模式就比struts更容易使用。

# Ruby

  • 《Introduction to Programming with Ruby》 (opens new window) Launch School 写的一本给新手看的入门书，涵盖最简单的编程概念和和练习及讲解。具体请看这个话题：https://ruby-china.org/topics/19914
  • 《Programming Ruby》 (opens new window) 著名的镐头书，推荐先看这本，内容活泼易懂。前半部分涵盖了 Ruby 的方方面面，后半部分是函数库参考，所以不要被它的厚度所吓到。
  • 《Ruby 编程语言》 (opens new window) 本书的作者之一是松本行弘，Ruby 的缔造者。Ruby 开发者的必看书籍，被称为 Ruby 版的 K&R（K&R 指《The C Programming Language》 (opens new window)，中文名《C 程序设计语言》，计算机史上的经典著作，其赞誉相当之高）。
  • 《Ruby 元编程》 (opens new window)
  • 《Metaprogramming Ruby》 (opens new window)
  • 《Ruby User’s Guide 简体中文版》 (opens new window)

# Rails

  • 《应用 Rails 进行敏捷 Web 开发》 (opens new window) 中文已更新至第四版，使用 Rails 3.0。 英文版本已经更新到 Agile Web Development with Rails 5，使用 Rails 5.0，书中部分内容已经重写，英语好的还是建议阅读 Rails 5.0 英文版。
  • 《Rails 实践》 (opens new window)
  • 《Ruby on Rails 实战圣经 Rails 3 Edition》 (opens new window)
  • 《Ruby on Rails 实战圣经 Rails 4 + Ruby 2.1 Edition》 (opens new window) 在线中文资料，作者*@*ihower (opens new window)，博客 (opens new window)。
  • 《Ruby on Rails Tutorial》 (opens new window) 在线中文版 (opens new window)，翻译：*@*andor_chen (opens new window)
  • 《Objects on Rails》 (opens new window)
